# 철자 검사 기능이 있는 채팅 애플리케이션

이 프로젝트는 Python의 `socket`과 `threading` 모듈을 사용하여 만든 간단한 클라이언트-서버 채팅 애플리케이션입니다. IRC와 유사한 프로토콜을 특징으로 하며, 선택적으로 영어 철자 검사 기능이 내장된 클라이언트를 제공합니다.

## 주요 기능

- 다중 사용자 채팅방
- 실시간 메시징
- 접속 중인 사용자 목록 표시
- 영어 철자 검사 및 추천 기능 (별도 클라이언트 제공)

## 실행 방법

1.  **서버 시작:**
    ```bash
    python chat/chat_server.py
    ```

2.  **클라이언트 실행:**
    -   **일반 클라이언트:**
        ```bash
        python chat/chat_client.py
        ```
    -   **철자 검사 기능이 있는 클라이언트:**
        ```bash
        python chat/chat_client_spell.py
        ```

## 프로젝트 구조

- `chat/`: 서버 및 클라이언트 소스 코드가 들어있습니다.
  - `chat_server.py`: 메인 채팅 서버입니다.
  - `chat_client.py`: 기본 채팅 클라이언트입니다.
  - `chat_client_spell.py`: 철자 검사 기능이 추가된 채팅 클라이언트입니다.
- `spell_checker/`: 철자 검사 모듈과 사전 파일이 들어있습니다.
  - `spell_checker_v2.py`: 핵심 철자 검사 로직을 담당합니다.
  - `words.txt`: 올바른 영어 단어 목록 파일입니다.
  - `en_full.txt`: 더 나은 추천을 위해 단어 빈도 정보가 포함된 파일입니다.

## 작동 방식

### 서버

서버(`chat_server.py`)는 멀티스레드 방식으로 구현되어 여러 클라이언트의 동시 접속을 처리할 수 있습니다. 새로운 클라이언트가 연결되면 각 클라이언트를 위한 별도의 스레드를 생성합니다. 통신에는 `LOGIN`, `MSG_ALL`, `QUIT`과 같은 IRC 스타일의 간단한 메시지 프로토콜을 사용합니다.

### 클라이언트

클라이언트(`chat_client.py`, `chat_client_spell.py`)는 서버에 접속하여 사용자가 메시지를 주고받을 수 있게 합니다. 별도의 스레드가 서버로부터 오는 메시지를 수신하여 화면에 표시합니다.

### 철자 검사기

철자 검사 기능이 있는 클라이언트(`chat_client_spell.py`)는 `spell_checker_v2.py` 모듈을 사용합니다. 사용자가 메시지를 입력하면 클라이언트는 다음 단계를 따릅니다:

1.  메시지에서 영어 단어를 추출합니다.
2.  각 단어가 올바른 단어 목록(`words.txt`)에 있는지 확인합니다.
3.  철자가 틀린 단어가 발견되면, 레벤슈타인 거리 알고리즘과 단어 빈도 맵(`en_full.txt`)을 사용하여 추천 단어를 생성합니다. 단어 빈도 정보는 더 정확하고 관련성 높은 추천을 제공하는 데 도움이 됩니다.
4.  사용자에게 수정된 메시지, 원본 메시지, 또는 직접 편집하여 보낼 수 있는 선택지를 제공합니다.
